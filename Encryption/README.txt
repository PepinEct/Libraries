Documentation

Key-class:
-Define a key object
-Parameters: -MINlen, -MAXlen, -AddChars (str)
-Key.make for creating a 'usable' Key
-Key.write: writes to 'KEY' file
-Key.load: loads key file

Encrypt function:
-Encrypt takes 2 arguments: Key object, Msg (str)
-Returns encrypted

DeEncrypt function:
-DeEncrypt takes 2 arguments: Key object, Msg (str)
-Returns DeEncrypted
